# ProjektApp

The project has to be run on a windows computer.

To start the application you need to go into Win32/Debug and start ProjektAppProject.exe.

To view the code without RAD Studio Community edition, open the source files ProjektAppUnit, CalendarUnit, NewEventUnit, EventDetailsUnit, ChangeViewUnit and TestUnit, but to get the full picture we would recommend RAD Studio and opening ProjectGroup1. Then choose the ProjektAppProject, which is a Delphi Project File. From RAD Studio you can run the application by pressing F9.

To run the test you need to open ProjectGroup1, and in RAD Studio choose the ProjectTest file. You should now be able to run the test with F9. You might have to run the test through the commandprompt to see if the test are successful, as the pop-up in RAD closes quickly.
